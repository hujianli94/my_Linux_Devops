# Nginx 正向代理配置

作用：客户端通过Nginx代理服务器来进行访问。

## 案例

**客户端添加代理：IP：8080**

```shell
server {
    # 端口
    listen       8080;
    # 地址
    server_name  localhost;
    # DNS解析地址
    resolver 8.8.8.8;
    # 代理参数
    location / {
        # $http_host就是我们要访问的主机名
        # $request_uri就是我们后面所加的参数
        proxy_pass http://$http_host$request_uri;
    }
}
```
