8.2. дјҒдёҡзә§Nginx+keepalived йӣҶзҫӨе®һжҲҳВ¶

иҙҹиҪҪеқҮиЎЎжҠҖжңҜеҜ№дәҺдёҖдёӘзҪ‘з«ҷе°Өе…¶жҳҜеӨ§еһӢзҪ‘з«ҷзҡ„webжңҚеҠЎеҷЁйӣҶзҫӨжқҘиҜҙжҳҜиҮіе…ійҮҚиҰҒзҡ„пјҒеҒҡеҘҪиҙҹиҪҪеқҮиЎЎжһ¶жһ„пјҢеҸҜд»Ҙе®һзҺ°ж•…йҡңиҪ¬з§»е’Ңй«ҳеҸҜз”ЁзҺҜеўғпјҢйҒҝе…ҚеҚ•зӮ№ж•…йҡңпјҢдҝқиҜҒзҪ‘з«ҷеҒҘеә·жҢҒз»ӯиҝҗиЎҢгҖӮ е…ідәҺиҙҹиҪҪеқҮиЎЎд»Ӣз»ҚпјҢ

з”ұдәҺдёҡеҠЎжү©еұ•пјҢзҪ‘з«ҷзҡ„и®ҝй—®йҮҸдёҚж–ӯеҠ еӨ§пјҢиҙҹиҪҪи¶ҠжқҘи¶Ҡй«ҳгҖӮзҺ°йңҖиҰҒеңЁwebеүҚз«Ҝж”ҫзҪ®nginxиҙҹиҪҪеқҮиЎЎ,еҗҢж—¶з»“еҗҲkeepalivedеҜ№еүҚз«Ҝnginxе®һзҺ°HAй«ҳеҸҜз”ЁгҖӮ

1пјүnginxиҝӣзЁӢеҹәдәҺMaster+Slave(worker)еӨҡиҝӣзЁӢжЁЎеһӢпјҢиҮӘиә«е…·жңүйқһеёёзЁіе®ҡзҡ„еӯҗиҝӣзЁӢз®ЎзҗҶеҠҹиғҪгҖӮеңЁMasterиҝӣзЁӢеҲҶй…ҚжЁЎејҸдёӢпјҢMasterиҝӣзЁӢж°ёиҝңдёҚиҝӣиЎҢдёҡеҠЎеӨ„зҗҶпјҢеҸӘжҳҜиҝӣиЎҢд»»еҠЎеҲҶеҸ‘пјҢд»ҺиҖҢиҫҫеҲ°MasterиҝӣзЁӢзҡ„еӯҳжҙ»й«ҳеҸҜйқ жҖ§пјҢSlave(worker)иҝӣзЁӢжүҖжңүзҡ„дёҡеҠЎдҝЎеҸ·йғҪ з”ұдё»иҝӣзЁӢеҸ‘еҮәпјҢSlave(worker)иҝӣзЁӢжүҖжңүзҡ„и¶…ж—¶д»»еҠЎйғҪдјҡиў«MasterдёӯжӯўпјҢеұһдәҺйқһйҳ»еЎһејҸд»»еҠЎжЁЎеһӢгҖӮ
2пјүKeepalivedжҳҜLinuxдёӢйқўе®һзҺ°VRRPеӨҮд»Ҫи·Ҝз”ұзҡ„й«ҳеҸҜйқ жҖ§иҝҗиЎҢд»¶гҖӮеҹәдәҺKeepalivedи®ҫи®Ўзҡ„жңҚеҠЎжЁЎејҸиғҪеӨҹзңҹжӯЈеҒҡеҲ°дё»жңҚеҠЎеҷЁе’ҢеӨҮд»ҪжңҚеҠЎеҷЁж•…йҡңж—¶IPзһ¬й—ҙж— зјқдәӨжҺҘгҖӮдәҢиҖ…з»“еҗҲпјҢеҸҜд»Ҙжһ„жһ¶еҮәжҜ”иҫғзЁіе®ҡзҡ„иҪҜд»¶LBж–№жЎҲгҖӮ

8.2.1. keepalivedй«ҳеҸҜз”Ёз®Җд»ӢВ¶

keepalivedжҳҜдёҖдёӘзұ»дјјдёҺlayer3гҖҒ4е’Ң7дәӨжҚўжңәеҲ¶зҡ„иҪҜд»¶пјҢkeepalivedиҪҜд»¶жңүдёӨз§ҚеҠҹиғҪпјҢеҲҶеҲ«жҳҜзӣ‘жҺ§жЈҖжҹҘгҖҒVRRP(иҷҡжӢҹи·Ҝз”ұеҷЁеҶ—дҪҷеҚҸи®®) keepalivedзҡ„дҪңз”ЁжҳҜжЈҖжөӢWebжңҚеҠЎеҷЁзҡ„зҠ¶жҖҒпјҢжҜ”еҰӮжңүдёҖеҸ°WebжңҚеҠЎеҷЁгҖҒMySQLжңҚеҠЎеҷЁе®•жңәжҲ–е·ҘдҪңеҮәзҺ°ж•…йҡңпјҢ keepalivedжЈҖжөӢеҲ°еҗҺпјҢдјҡе°Ҷж•…йҡңзҡ„WebжңҚеҠЎеҷЁжҲ–иҖ…MySQLжңҚеҠЎеҷЁд»Һзі»з»ҹдёӯеү”йҷӨпјҢ еҪ“жңҚеҠЎеҷЁе·ҘдҪңжӯЈеёёеҗҺkeepalivedиҮӘеҠЁе°ҶжңҚеҠЎеҷЁеҠ е…ҘеҲ°жңҚеҠЎеҷЁзҫӨдёӯпјҢиҝҷдәӣе·ҘдҪңе…ЁйғЁиҮӘеҠЁе®ҢжҲҗпјҢ дёҚйңҖиҰҒдәәе·Ҙе№Іж¶үпјҢйңҖиҰҒдәәе·ҘеҒҡзҡ„еҖјжҳҜдҝ®еӨҚж•…йҡңзҡ„Webе’ҢMySQLжңҚеҠЎеҷЁгҖӮlayer3гҖҒ4гҖҒ7е·ҘдҪңеңЁTCP/IPеҚҸи®®ж Ҳзҡ„IPеұӮгҖҒдј иҫ“еұӮгҖҒеә”з”ЁеұӮпјҢе®һзҺ°еҺҹзҗҶдёәпјҡ

layer3пјҡkeepalivedдҪҝз”Ёlayer3зҡ„ж–№ејҸе·ҘдҪңж—¶пјҢkeepalivedдјҡе®ҡжңҹеҗ‘жңҚеҠЎеҷЁзҫӨдёӯзҡ„жңҚеҠЎеҷЁеҸ‘йҖҒдёҖдёӘICMPж•°жҚ®еҢ…пјҢеҰӮжһңеҸ‘зҺ°жҹҗеҸ°жңҚеҠЎзҡ„IPең°еқҖж— жі•pingйҖҡпјҢkeepalivedдҫҝжҠҘе‘ҠиҝҷеҸ°жңҚеҠЎеҷЁеӨұж•ҲпјҢе№¶е°Ҷе®ғд»ҺжңҚеҠЎеҷЁйӣҶзҫӨдёӯеү”йҷӨгҖӮlayer3зҡ„ж–№ејҸжҳҜд»ҘжңҚеҠЎеҷЁзҡ„IPең°еқҖжҳҜеҗҰжңүж•ҲдҪңдёәжңҚеҠЎеҷЁе·ҘдҪңжҳҜеҗҰжӯЈеёёзҡ„ж ҮеҮҶ layer4пјҡlayer4дё»иҰҒд»ҘTCPз«ҜеҸЈзҡ„зҠ¶жҖҒжқҘеҶіе®ҡжңҚеҠЎеҷЁе·ҘдҪңжҳҜеҗҰжӯЈеёёгҖӮдҫӢеҰӮWebжңҚеҠЎз«ҜеҸЈдёҖиҲ¬дёә80пјҢеҰӮжһңkeepalivedжЈҖжөӢеҲ°80з«ҜеҸЈжІЎжңүеҗҜеҠЁпјҢеҲҷkeepalivedжҠҠиҝҷеҸ°жңҚеҠЎеҷЁд»ҺжңҚеҠЎеҷЁйӣҶзҫӨдёӯеү”йҷӨ layer7пјҡlayer7е·ҘдҪңеңЁеә”з”ЁеұӮпјҢkeepalivedе°Ҷж №жҚ®з”ЁжҲ·зҡ„и®ҫе®ҡжЈҖжҹҘжңҚеҠЎеҷЁзҡ„иҝҗиЎҢжҳҜеҗҰжӯЈеёёпјҢеҰӮжһңдёҺз”ЁжҲ·зҡ„и®ҫе®ҡдёҚзӣёз¬ҰпјҢеҲҷkeepalivedе°ҶжҠҠжңҚеҠЎеҷЁд»ҺжңҚеҠЎеҷЁйӣҶзҫӨдёӯеү”йҷӨ

VRRPеҚҸи®®пјҡ

VRRPе…Ёз§° Virtual Router Redundancy ProtocolпјҢеҚі иҷҡжӢҹи·Ҝз”ұеҶ—дҪҷеҚҸи®®гҖӮ
еҸҜд»Ҙи®Өдёәе®ғжҳҜе®һзҺ°и·Ҝз”ұеҷЁй«ҳеҸҜз”Ёзҡ„е®№й”ҷеҚҸи®®пјҢеҚіе°ҶNеҸ°жҸҗдҫӣзӣёеҗҢеҠҹиғҪзҡ„и·Ҝз”ұеҷЁз»„жҲҗдёҖдёӘи·Ҝз”ұеҷЁз»„(Router Group)пјҢ
иҝҷдёӘз»„йҮҢйқўжңүдёҖдёӘmasterе’ҢеӨҡдёӘbackupпјҢдҪҶеңЁеӨ–з•ҢзңӢжқҘе°ұеғҸдёҖеҸ°дёҖж ·пјҢжһ„жҲҗиҷҡжӢҹи·Ҝз”ұеҷЁпјҢ
жӢҘжңүдёҖдёӘиҷҡжӢҹIPпјҲvipпјҢд№ҹе°ұжҳҜи·Ҝз”ұеҷЁжүҖеңЁеұҖеҹҹзҪ‘еҶ…е…¶д»–жңәеҷЁзҡ„й»ҳи®Өи·Ҝз”ұпјүпјҢ
еҚ жңүиҝҷдёӘIPзҡ„masterе®һйҷ…иҙҹиҙЈARPзӣёеә”е’ҢиҪ¬еҸ‘IPж•°жҚ®еҢ…пјҢз»„дёӯзҡ„е…¶е®ғи·Ҝз”ұеҷЁдҪңдёәеӨҮд»Ҫзҡ„и§’иүІеӨ„дәҺеҫ…е‘ҪзҠ¶жҖҒгҖӮ
masterдјҡеҸ‘з»„ж’ӯж¶ҲжҒҜпјҢеҪ“backupеңЁи¶…ж—¶ж—¶й—ҙеҶ…ж”¶дёҚеҲ°vrrpеҢ…ж—¶е°ұи®Өдёәmasterе®•жҺүдәҶпјҢ
иҝҷж—¶е°ұйңҖиҰҒж №жҚ®VRRPзҡ„дјҳе…Ҳзә§жқҘйҖүдёҫдёҖдёӘbackupеҪ“masterпјҢдҝқиҜҒи·Ҝз”ұеҷЁзҡ„й«ҳеҸҜз”ЁгҖӮ

еңЁVRRPеҚҸи®®е®һзҺ°йҮҢпјҢиҷҡжӢҹи·Ҝз”ұеҷЁдҪҝз”Ё 00-00-5E-00-01-XX дҪңдёәиҷҡжӢҹMACең°еқҖпјҢ
XXе°ұжҳҜе”ҜдёҖзҡ„ VRID пјҲVirtual Router IDentifierпјүпјҢиҝҷдёӘең°еқҖеҗҢдёҖж—¶й—ҙеҸӘжңүдёҖдёӘзү©зҗҶи·Ҝз”ұеҷЁеҚ з”ЁгҖӮ
еңЁиҷҡжӢҹи·Ҝз”ұеҷЁйҮҢйқўзҡ„зү©зҗҶи·Ҝз”ұеҷЁз»„йҮҢйқўйҖҡиҝҮеӨҡж’ӯIPең°еқҖ 224.0.0.18 жқҘе®ҡж—¶еҸ‘йҖҒйҖҡе‘Ҡж¶ҲжҒҜгҖӮ
жҜҸдёӘRouterйғҪжңүдёҖдёӘ 1-255 д№Ӣй—ҙзҡ„дјҳе…Ҳзә§еҲ«пјҢзә§еҲ«жңҖй«ҳзҡ„пјҲhighest priorityпјүе°ҶжҲҗдёәдё»жҺ§пјҲmasterпјүи·Ҝз”ұеҷЁгҖӮ
йҖҡиҝҮйҷҚдҪҺmasterзҡ„дјҳе…ҲжқғеҸҜд»Ҙи®©еӨ„дәҺbackupзҠ¶жҖҒзҡ„и·Ҝз”ұеҷЁжҠўеҚ пјҲpro-emptпјүдё»и·Ҝз”ұеҷЁзҡ„зҠ¶жҖҒпјҢ
дёӨдёӘbackupдјҳе…Ҳзә§зӣёеҗҢзҡ„IPең°еқҖиҫғеӨ§иҖ…дёәmasterпјҢжҺҘз®ЎиҷҡжӢҹIPгҖӮ

keepalivedдёҺheartbeat/corosyncзӯүжҜ”иҫғпјҡ HeartbeatгҖҒCorosyncгҖҒKeepalivedиҝҷдёүдёӘйӣҶзҫӨз»„д»¶жҲ‘д»¬еҲ°еә•йҖүе“ӘдёӘеҘҪе‘ўпјҹ йҰ–е…ҲиҰҒиҜҙжҳҺзҡ„жҳҜпјҢHeartbeatгҖҒCorosyncжҳҜеұһдәҺеҗҢдёҖзұ»еһӢпјҢKeepalivedдёҺHeartbeatгҖҒCorosyncпјҢж №жң¬дёҚжҳҜеҗҢдёҖзұ»еһӢзҡ„гҖӮ

KeepalivedдҪҝз”Ёзҡ„vrrpеҚҸи®®ж–№ејҸпјҢиҷҡжӢҹи·Ҝз”ұеҶ—дҪҷеҚҸи®® (Virtual Router Redundancy ProtocolпјҢз®Җз§°VRRP)пјӣ HeartbeatжҲ–CorosyncжҳҜеҹәдәҺдё»жңәжҲ–зҪ‘з»ңжңҚеҠЎзҡ„й«ҳеҸҜз”Ёж–№ејҸпјӣ з®ҖеҚ•зҡ„иҜҙе°ұжҳҜпјҢKeepalivedзҡ„зӣ®зҡ„жҳҜжЁЎжӢҹи·Ҝз”ұеҷЁзҡ„й«ҳеҸҜз”ЁпјҢHeartbeatжҲ–Corosyncзҡ„зӣ®зҡ„жҳҜе®һзҺ°Serviceзҡ„й«ҳеҸҜз”ЁгҖӮ жүҖд»ҘдёҖиҲ¬KeepalivedжҳҜе®һзҺ°еүҚз«Ҝй«ҳеҸҜз”ЁпјҢеёёз”Ёзҡ„еүҚз«Ҝй«ҳеҸҜз”Ёзҡ„з»„еҗҲжңү

е®һзҺ°MySQLжңҚеҠЎеҷЁзҡ„й«ҳеҸҜз”ЁгҖӮжҖ»з»“дёҖдёӢпјҢKeepalivedдёӯе®һзҺ°иҪ»йҮҸзә§зҡ„й«ҳеҸҜз”ЁпјҢдёҖиҲ¬з”ЁдәҺеүҚз«Ҝй«ҳеҸҜз”ЁпјҢ
дё”дёҚйңҖиҰҒе…ұдә«еӯҳеӮЁпјҢдёҖиҲ¬еёёз”ЁдәҺдёӨдёӘиҠӮзӮ№зҡ„й«ҳеҸҜз”ЁгҖӮиҖҢHeartbeat(жҲ–Corosync)дёҖиҲ¬з”ЁдәҺжңҚеҠЎзҡ„й«ҳеҸҜз”ЁпјҢ
дё”йңҖиҰҒе…ұдә«еӯҳеӮЁпјҢдёҖиҲ¬з”ЁдәҺеӨҡиҠӮзӮ№зҡ„й«ҳеҸҜз”ЁгҖӮиҝҷдёӘй—®йўҳжҲ‘д»¬иҜҙжҳҺзҷҪдәҶгҖӮ

йӮЈheartbaetдёҺcorosyncеҸҲеә”иҜҘйҖүжӢ©е“ӘдёӘеҘҪпјҹ

дёҖиҲ¬з”ЁcorosyncпјҢеӣ дёәcorosyncзҡ„иҝҗиЎҢжңәеҲ¶жӣҙдјҳдәҺheartbeatпјҢе°ұиҝһд»ҺheartbeatеҲҶзҰ»еҮәжқҘзҡ„pacemakerйғҪиҜҙеңЁд»ҘеҗҺзҡ„ејҖеҸ‘еҪ“дёӯжӣҙеҖҫеҗ‘дәҺcorosyncпјҢжүҖд»ҘзҺ°еңЁcorosync+pacemakerжҳҜжңҖдҪіз»„еҗҲгҖӮ

еҸҢжңәй«ҳеҸҜз”ЁдёҖиҲ¬жҳҜйҖҡиҝҮиҷҡжӢҹIPпјҲйЈҳз§»IPпјүж–№жі•жқҘе®һзҺ°зҡ„пјҢеҹәдәҺLinux/Unixзҡ„IPеҲ«еҗҚжҠҖжңҜгҖӮ еҸҢжңәй«ҳеҸҜз”Ёж–№жі•зӣ®еүҚеҲҶдёәдёӨз§Қпјҡ

1пјүеҸҢжңәдё»д»ҺжЁЎејҸпјҡеҚіеүҚз«ҜдҪҝз”ЁдёӨеҸ°жңҚеҠЎеҷЁпјҢдёҖеҸ°дё»жңҚеҠЎеҷЁе’ҢдёҖеҸ°зғӯеӨҮжңҚеҠЎеҷЁпјҢжӯЈеёёжғ…еҶөдёӢпјҢдё»жңҚеҠЎеҷЁз»‘е®ҡдёҖдёӘе…¬зҪ‘иҷҡжӢҹIPпјҢжҸҗдҫӣиҙҹиҪҪеқҮиЎЎжңҚеҠЎпјҢзғӯеӨҮжңҚеҠЎеҷЁеӨ„дәҺз©әй—ІзҠ¶жҖҒпјӣеҪ“дё»жңҚеҠЎеҷЁеҸ‘з”ҹж•…йҡңж—¶пјҢзғӯеӨҮжңҚеҠЎеҷЁжҺҘз®Ўдё»жңҚеҠЎеҷЁзҡ„е…¬зҪ‘иҷҡжӢҹIPпјҢжҸҗдҫӣиҙҹиҪҪеқҮиЎЎжңҚеҠЎпјӣдҪҶжҳҜзғӯеӨҮжңҚеҠЎеҷЁеңЁдё»жңәеҷЁдёҚеҮәзҺ°ж•…йҡңзҡ„ж—¶еҖҷпјҢж°ёиҝңеӨ„дәҺжөӘиҙ№зҠ¶жҖҒпјҢеҜ№дәҺжңҚеҠЎеҷЁдёҚеӨҡзҡ„зҪ‘з«ҷпјҢиҜҘж–№жЎҲдёҚз»ҸжөҺе®һжғ гҖӮ
2пјүеҸҢжңәдё»дё»жЁЎејҸпјҡеҚіеүҚз«ҜдҪҝз”ЁдёӨеҸ°иҙҹиҪҪеқҮиЎЎжңҚеҠЎеҷЁпјҢдә’дёәдё»еӨҮпјҢдё”йғҪеӨ„дәҺжҙ»еҠЁзҠ¶жҖҒпјҢеҗҢж—¶еҗ„иҮӘз»‘е®ҡдёҖдёӘе…¬зҪ‘иҷҡжӢҹIPпјҢжҸҗдҫӣиҙҹиҪҪеқҮиЎЎжңҚеҠЎпјӣеҪ“е…¶дёӯдёҖеҸ°еҸ‘з”ҹж•…йҡңж—¶пјҢеҸҰдёҖеҸ°жҺҘз®ЎеҸ‘з”ҹж•…йҡңжңҚеҠЎеҷЁзҡ„е…¬зҪ‘иҷҡжӢҹIPпјҲиҝҷж—¶з”ұйқһж•…йҡңжңәеҷЁдёҖеҸ°иҙҹжӢ…жүҖжңүзҡ„иҜ·жұӮпјүгҖӮиҝҷз§Қж–№жЎҲпјҢз»ҸжөҺе®һжғ пјҢйқһеёёйҖӮеҗҲдәҺеҪ“еүҚжһ¶жһ„зҺҜеўғгҖӮ

8.2.3. 1гҖҒеҺҹзҗҶеҸҠзҺҜеўғВ¶

NginxиҙҹиҪҪеқҮиЎЎдёҖиҲ¬дҪҚдәҺж•ҙдёӘжһ¶жһ„зҡ„жңҖеүҚз«ҜжҲ–иҖ…дёӯй—ҙеұӮпјҢеҰӮжһңдёәжңҖеүҚз«Ҝж—¶еҚ•еҸ°nginxдјҡеӯҳеңЁеҚ•зӮ№ж•…йҡңпјҢ
дёҖеҸ°nginxе®•жңәпјҢдјҡеҪұе“Қз”ЁжҲ·еҜ№ж•ҙдёӘзҪ‘з«ҷзҡ„и®ҝй—®гҖӮеҰӮжһңйңҖиҰҒеҠ е…ҘnginxеӨҮд»ҪжңҚеҠЎеҷЁпјҢnginxдё»жңҚеҠЎеҷЁдёҺеӨҮд»ҪжңҚеҠЎеҷЁд№Ӣй—ҙеҪўжҲҗй«ҳеҸҜз”ЁпјҢ
дёҖж—ҰеҸ‘зҺ°nginxдё»е®•жңәпјҢиғҪеӨҹеҝ«йҖҹе°ҶзҪ‘з«ҷеҲҮжҚўиҮіеӨҮд»ҪжңҚеҠЎеҷЁгҖӮ
