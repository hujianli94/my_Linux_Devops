2.2. 华为防火墙技术基础知识¶

2.2.1. 基础知识¶

安全区域¶

通过把接口划分到不同的安全区域中，就可以在防火墙上划分出不通的网络。

例如：我们把接口1和接口2划分到安全区域A中。
把接口3划分到安全域B中
把接口4划分到安全区域C中，
这样防火墙就存在三个安全区域，对应三个网络。

华为防火墙上已经默认为大家提供了三个安全域。分别是 Trust、DMZ和Untrust。

① Trust 区域： 该区域内网络的受限任程度高，通常用来定义内部用户所在的网络。
② DMZ 区域：该区域内网络的受信任程度中等，通常用来定义内部服务器所在的网络。
③ Untrust 区域：该区域代表的是不受信任的网络，通常用来定义Internet等不安全的网络。

如果接口1和接口2连接的是内部用户，我们就把这两个接口划分到Trust区域中；

接口3连接的是内部服务器，我们就把这个接口划分到DMZ区域中，

接口4连接的是Internet，将它划分到Untrust区域。

例如：

· 当内部网络用户访问Internet的时候，走的线路是 Trust 区域 —> Untrust 区域

· 当Internet上的用户访问内部服务器时，报文在防火墙的线路是 Untrust 区域 —> Trust 区域

防火墙提供Local区域，代表防火墙本身。凡是由防火墙主动发出的报文均可认为是从Local区域中发出。 凡是需要防火墙响应并处理（而不是转发）的报文可认为是由Local区域接口。

Local区域中不能添加任何接口，防火墙上所有接口本身都隐含于Local区域，报文通过接口去往某个网络时，目地安全区是接口所在的安全区域。
报文通过接口到达防火墙本身时，目地安全区域是Local区域。这样既可以使每个接口连接的其他设备能够访问防火墙自身，也能明确Local区域和各个安全区域的域间关系，一举两得。

报文在安全区域之间的流动方向¶

对于默认的安全区域，它们的安全级别是固定的：
Local区域的安全级别是100，
Trust区域的安全级别是85，
DMZ区域的安全级别是50，
Untrust区域的安全级别是5.


报文从低级别的安全区域向高级版的安全区域流动时为入方向（Inbound）

报文从高级别的安全区域向低级版的安全区域流动时为入方向（Outbound）


确定目的安全区域时分两种情况，

三层模式下：防火墙查路由表确定报文从哪个接口发出，该接口所属的安全域就是报文的目的安全区域。
二层模式下：防火墙通过查MAC地址转发表确定报文将要从哪个接口发出，该接口所属的安全区域就是报文的目的安全区域。

源安全区域和目的安全区域确定后，就可以指定报文是在哪两个安全区域之间流动了。

VPN场景中放好收到的是封装的报文，将报文解封装后得到原始报文，然后查找路由表来确定目的安全区域，报文将要从哪个接口发出，
该接口所属的安全区域就是报文的目的安全区域。

安全区域的配置¶

firewall zone name test                 //创建安全区域test
set priority 10                         // 将安全级别设置为10
add interface GigabitEthernet 0/0/1      // 将接口GE 0/0/1加入安全区域

通过子接口将PC1和PC2放入两个不同的zone¶

interface GigabitEthernet 0/0/1.10
vlan-type dotlq 10
ip address 192.168.10.1 24
quit

interface GigabitEthernet 0/0/1.20
vlan-type dotlq 20
ip address 192.168.20.1 24
quit

zone name trustl
set priority 10
add interface GigabitEthernet 0/0/1.10
quit

zone name trust2
set priority 120
add interface GigabitEthernet 0/0/1.20
quit

完成上述配置后
PC1被划分到trustl安全区域，
PC2被划分到trust2安全区域，此时就可以对PC1和PC2的报文进行控制

通过Vlanif的方式对PC1和PC2进行zone的划分¶

vlan 10
quit
interface V1anif 10
quit

vlan 20
quit
interface VIanif 20
quit

interface GigabitEthernet 0/0/1
portswitch
port link-type trunk
port trunk permit vlan 10 20
quit

firewall zone name trust1
set priority 10
add interface V1anif 10
quit


firewall zone name trust2
set priority 20
add interface V1anif 20
quit

状态检测¶

display firewall session table verbose      //防火墙上查看状态检测会话

undo display firewall session link-state check  //关闭状态检测功能

display zone        //查看防火墙上安全区域的配置情况
display firewall statistic system discard       //查看防火墙的丢包统计信息
